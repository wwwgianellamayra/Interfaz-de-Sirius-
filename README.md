# SIRIUS · Estación Terrena

Interfaz web local para visualizar la telemetría del CubeSat SIRIUS, desarrollada para la Competencia Nacional de Pequeños Satélites.

## Requisitos

- Python 3.10 o superior
- Visual Studio Code
- Git

## Instalación en Windows

```powershell
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python app.py
